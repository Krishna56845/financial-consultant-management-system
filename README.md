# Financial Consultant Management System

A web-based application developed using Flask to manage financial consultants, clients, appointments, and payments efficiently.

## Features
- Financial consultant and client management (CRUD operations)
- Appointment booking system
- Payment tracking with multiple payment methods
- Dashboard with analytics and insights
- AI-based client risk advisory (rule-based)

## Technologies Used
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML, CSS, Bootstrap

## Project Objective
To provide a centralized platform for managing financial consultants and their clients, improving efficiency, data organization, and decision-making.
